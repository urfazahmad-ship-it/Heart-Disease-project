# Heart-Disease-project
Machine Learning project to predict heart disease using patient health data, comparing Logistic Regression, KNN, Naive Bayes, Decision Tree, and SVM models.

This project analyzes a heart disease dataset and builds machine learning models to predict whether a patient has heart disease based on clinical features such as age, resting blood pressure, cholesterol, and maximum heart rate.
Workflow:

Exploratory Data Analysis (EDA) with visualizations (distribution plots, correlation heatmap)
Data cleaning (handling invalid zero values in Cholesterol and RestingBP)
Feature encoding (one-hot encoding for categorical variables)
Feature scaling using StandardScaler
Model training and comparison across 5 algorithms:

Logistic Regression
K-Nearest Neighbors (KNN)
Naive Bayes
Decision Tree
Support Vector Machine (SVM)

Model evaluation using Accuracy and F1 Score
Tech stack: Python, Pandas, NumPy, Seaborn, Matplotlib, Scikit-learn
